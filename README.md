A modern, full-stack ERP (Enterprise Resource Planning) and Dashboard application designed specifically for single shop management. It seamlessly tracks Income, Expenses, Inventory, Products, Purchases, and Udhar (Credit) Ledgers.


## 🛠️ Features

- **Dashboard**: High-level metrics, daily/monthly profit analysis, and visual charts.
- **Inventory Management**: Track stock levels, set minimum stock alerts, and monitor purchase/selling prices.
- **Ledger (Udhar) Tracking**: Efficiently manage customer credits, due dates, and partial/full payments.
- **Income & Expense Tracking**: Keep daily logs of all financial transactions.
- **Role-based Auth**: Secure login mechanism via JWT.

##  Technology Stack

- **Frontend**: React 19, Vite, Tailwind CSS (v4), Recharts (for analytics), Lucide React (icons).
- **Backend**: Node.js, Express.js.
- **Database**: PostgreSQL (via `pg` driver).

For advanced deployment, database architecture, and offline/desktop app conversion strategies, please refer to our full [Documentation](documentation.md).
